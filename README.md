# monday-kai-automations

Automations for monday.com workflows, built with n8n and integrated with GitHub and Gmail.

## Overview

This project contains automation workflows and scripts for managing monday.com boards, items, and notifications.

## Structure

```
monday-kai-automations/
├── workflows/        # n8n workflow exports
├── scripts/          # Utility scripts
└── docs/             # Documentation
```

## Getting Started

1. Clone the repository
2. Set up your environment variables (see `.env.example`)
3. Import workflows from the `workflows/` directory into n8n

## Environment Variables

Copy `.env.example` to `.env` and fill in your values:

```bash
cp .env.example .env
```
